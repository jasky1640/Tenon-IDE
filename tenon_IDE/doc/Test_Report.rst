================ ====================================================================================================================================================================
项目/软件        tenon_IDE

功能模块         编辑器

开发人员         韩柳彤，韦建，杨家琪。

Ubuntu 16.04.6；

                 3. Eclipse version "4.11.0"。

                 将TenonEditor_0.9.0.jar(本次被测试版本)放入Eclipse安装目录下的plugins或dropin文件夹;测试所用文件arith_scalar.t，放入该测试文档同目录;
                 测试所用文件hello.t放入/tmp目录。
